# Household Notes

- use this file for durable household-system notes that do not belong in the live weekly state
- for now, automate only what can be honestly derived from recorded dates, status, effort, and manual threshold checks
- wooden utensil maintenance should stay small:
  - run a monthly-to-bimonthly check
  - oil only the items that actually look dry, faded, rough, chalky, or unusually thirsty
  - wash and fully dry first
  - use a thin coat of food-safe mineral oil or board butter
  - let it sit for a few hours or overnight, then wipe off excess
  - the goal is steady maintenance on high-use items, not surprise whole-batch restoration
